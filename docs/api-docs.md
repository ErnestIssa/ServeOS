# API Docs

This is a placeholder for ServeOS API documentation.

## Conventions (planned)

- REST for CRUD + admin workflows
- WebSocket (or SSE) for real-time order/reservation updates
- JWT/session tokens issued by `auth-service`
- Gateway validates auth and routes to domain services
